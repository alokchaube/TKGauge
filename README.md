# TKGauge

iOS UI component to display gauge

## Installation

* You need [cocoapods](http://cocoapods.org) lib manager
* Edit your `Podfile` and add

```ruby
dependency 'TKGauge', :podspec => 'https://raw.github.com/xslim/TKGauge/master/TKGauge.podspec'
```
